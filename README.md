# Sales & Revenue Analysis Dashboard

## 📌 Project Overview
This project presents an interactive Sales & Revenue Analysis Dashboard developed using Power BI. The dashboard helps analyze sales performance, profitability, customer segments, product categories, and regional performance through interactive visualizations and KPI tracking.

## 🎯 Objective
To build a business intelligence dashboard that enables:
- Sales and revenue monitoring
- KPI tracking
- Product performance analysis
- Regional performance analysis
- Interactive data exploration using filters and slicers

## 🛠️ Tools & Technologies
- Power BI Desktop
- CSV Dataset (Sample Superstore)
- Data Visualization
- Business Intelligence & Analytics

## 📊 Key Performance Indicators (KPIs)
- Total Sales
- Total Profit
- Total Quantity Sold
- Average Discount

## 📈 Dashboard Features
### Sales Analysis
- Sales by Category
- Sales by Segment
- Sales Trend Analysis

### Profit Analysis
- Profit by Region
- Profit by Category
- Top Profit-Generating Sub-Categories

### Product Performance
- Top Sales by Sub-Category
- Top Profit by Sub-Category

### Advanced Visualizations
- Scatter Plot (Sales vs Profit)
- Geographic Map Analysis
- Data Table Summary
- Gauge Chart for Sales KPI

### Interactive Filtering
- Category Slicer
- Region Slicer
- Segment Slicer
- Ship Mode Slicer

## 📋 Business Insights Generated
- Identified top-performing product categories and sub-categories.
- Analyzed profit contribution across different regions.
- Evaluated sales distribution across customer segments.
- Tracked sales trends over time.
- Explored geographical sales performance using map visualization.
